# compra_venta_app.py

Este proyecto es una aplicación simple de compra y venta en Python.

## Estructura
- `main.py`: Script principal de la aplicación.
- `.github/copilot-instructions.md`: Instrucciones para Copilot.

## Uso
Ejecuta el script principal:

```bash
python main.py
```

¡Bienvenido a la aplicación de compra y venta!
